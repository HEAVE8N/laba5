## Задание 1







### Задача 1







### Текст задачи







#### Составить программу нахождения числа, которое образуется из данного
натурального числа при записи его цифр в обратном порядке. Например, для
числа 1234 получаем ответ 4321.







### Алгоритм решения



######



#### Программа на Prolog выполняет реверс числа через хвостовую рекурсию с аккумулятором. Предикат go считывает N и запускает rev, где на каждом шаге от N отделяется последняя цифра (mod 10) и добавляется к результату (Acc * 10), а само число сокращается (// 10). Когда N обнуляется, накопленное значение выводится на экран как готовый результат.

# Задание 2







### Задача 1



###



### Текст задачи







#### В заданном списке определить максимальное количество подряд идущих
положительных чисел. 







### Алгоритм решения







#### Алгоритм на Prolog находит длину самой длинной непрерывной серии положительных чисел в списке. Предикат start считывает список и передает его функции f с двумя аккумуляторами: C (текущая длина серии) и B (максимальная найденная длина). Если очередной элемент H > 0, счетчик C увеличивается; если же встречается неположительное число, текущая длина C сравнивается с максимумом B, после чего C обнуляется. Когда список заканчивается, программа через max(C, B) вычисляет финальный результат и выводит его на экран.

### Алгоритм решения







#### Этот алгоритм на Prolog реализует разность двух списков (

), удаляя из первого списка все элементы, которые присутствуют во втором. Предикат m (member) проверяет наличие элемента в списке, а основной предикат c (collect) рекурсивно обходит список A: если текущий элемент H найден в списке B, он пропускается, в противном случае — добавляется в результирующий список R. Предикат go считывает оба списка, проверяет корректность ввода и выводит элементы списка A, которых нет в B, сохраняя их исходный порядок.
